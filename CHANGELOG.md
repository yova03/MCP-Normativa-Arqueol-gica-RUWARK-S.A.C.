# Changelog

Todas las versiones relevantes de `mcp-normativa-ruwark` se documentan aquí.

## [2026.3.1] - 2026-03-09

### Added

- Suite de pruebas de humo para índice, búsqueda y herramientas principales.
- Workflow de GitHub Actions para pruebas y build del paquete.
- Documentación de instalación desde GitHub y configuración en clientes MCP.
- Soporte para `NORMATIVA_RUWARK_KB_DIR` como override opcional de la knowledge base.

### Changed

- Se excluye el `README.md` raíz de la knowledge base del índice de búsqueda.
- Se mejoró el `pyproject.toml` con metadatos, extras de desarrollo y configuración de pruebas.

## [2026.3.0] - 2026-03-09

### Added

- Servidor MCP inicial con herramientas, recursos y prompts para normativa arqueológica del Perú.
- Base documental en Markdown para RIA, Ley 28296, TUPA, TUO, procedimientos y decretos supremos.
