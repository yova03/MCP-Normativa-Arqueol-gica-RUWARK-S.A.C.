# Knowledge Base de Normativa Arqueológica

Base documental del MCP `mcp-normativa-ruwark`.

## Resumen

- 147 documentos normativos indexados.
- 113 decretos supremos del Ministerio de Cultura entre 2015 y 2025.
- Procedimientos incluidos: PMAR, PEA, DAS, OTF y CIRAS.
- Normativa base incluida: RIA, Ley N.° 28296, TUPA 2025 y TUO Ley N.° 27444.
