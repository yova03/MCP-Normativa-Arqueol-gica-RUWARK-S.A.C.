# Flujo del Diagnóstico Arqueológico de Superficie (DAS)

> **Base legal:** No regulado directamente en el RIA — procedimiento interno DDC
> **Tasa TUPA:** No aplica

---

## 1. ¿Qué es el DAS?

El Diagnóstico Arqueológico de Superficie es un **informe técnico de evaluación visual** del terreno sin excavación, realizado por un arqueólogo habilitado. Sirve como evaluación preliminar para determinar si existen indicios de presencia arqueológica.

## 2. ¿Cuándo se usa?

- Como paso previo informal antes de solicitar CIRAS o PEA.
- Cuando el cliente necesita un **dictamen técnico rápido** sobre un terreno.
- Para sustentar la viabilidad arqueológica de un proyecto ante inversionistas.
- Como parte de Estudios de Impacto Ambiental (EIA).

## 3. Flujo del Proceso

```
1. Cliente solicita diagnóstico de terreno
2. Verificar ubicación y antecedentes en SIGDA
3. Planificar visita de campo
4. Realizar inspección de superficie (sin excavación)
5. Documentar: fotografías, coordenadas, observaciones
6. Consultar base de datos de sitios arqueológicos
7. Elaborar informe técnico DAS
8. Entregar al cliente con recomendaciones
```

## 4. Contenido del Informe DAS

1. **Datos generales** — Ubicación, área, proyecto
2. **Antecedentes** — Consulta SIGDA, estudios previos
3. **Descripción del terreno** — Topografía, geología, vegetación
4. **Observaciones de campo** — Evidencia en superficie (cerámica, líticos, muros, etc.)
5. **Registro fotográfico** — Con coordenadas y orientación
6. **Conclusiones** — ¿Hay indicios arqueológicos?
7. **Recomendaciones** — Solicitar CIRAS, PEA, o proceder sin restricción

## 5. Diferencia con otras intervenciones

| Aspecto | DAS | CIRAS | PEA |
|---------|-----|-------|-----|
| Excavación | No | No (inspección ocular) | Sí |
| Tasa TUPA | No | Sí | Sí |
| R.D. requerida | No | No (es el resultado) | Sí |
| Emitido por | Consultor privado | MinCul | Consultor autorizado |
| Valor legal | Referencial | Oficial | Oficial |
